Info for stage 1-1
  • New blockFat Block
    The color of this type of block is pink. It stands for the fat inside Fallen_Cat's body. If the table of the stage have not any fat block, You will beat the stage.
  • New blockExercise Block
    The color of this type of block is blue. It stands for the lose weight plans of Fallen_Cat. You can merge exercise blocks with fat blocks to Eliminate them and help Fallen_Cat to lose weight.
  • How to playuse your mouse to select blocks and use WASD (If you are a mobile player, please use the keyboard in the left-up corner) on the keyboard to move blocks in the grid. Your goal is eliminate all fat blocks in the table. If you have no exercise block but fat blocks, you'll fail the stage.
Info for stage 1-2
  • HintYou can merge 2 exercise blocks together and their values will be stacked.
Info for stage 1-4
  • HintIf you don't take exercise or eat too much during a day, your body will get fatter.
    The value of exercise blocks and fat blocks WILL BE INCREASED BY 1 if you move them to a empty slot.
Info for stage 1-5
  • HintTry to use less clicks and less moves to finish the stage. Then your score will get higher.
Info for stage 1-6
  • HintYou cannot merge 2 fat blocks together.
Info for stage 1-9
  • HintTry to form a long path of blocks and combine blocks along the path at once. This will help reduce the number of clicks.
Info for stage 2-1
  • New blockBarrier Block
    A kind of block which cannot be moved or merged with any other blocks. It's useless.
Info for stage 3-1
  • New blockMultiply Block
    A kind of block which can multiply the value of exercise blocks or fat blocks when they moved onto it. But itself cannot be moved to any place.
Info for stage 3-4
  • New blockDelete Block
    A kind of block which can eliminate exercise blocks or fat blocks when they moved onto it. But itself cannot be moved to any place.
  • HintYou can reach a weight below 45 kilograms in some stages. Try to reach lower weight in these stage and get a higher score!
