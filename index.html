Info for stage 1-1
  • New blockFat Block
    The color of this type of block is pink. It stands for the fat inside Fallen_Cat's body. If the table of the stage doesn't contain any fat blocks, You will beat the stage.
  • New blockExercise Block
    The color of this type of block is blue. It stands for the lose weight plans of Fallen_Cat. You can merge exercise blocks with fat blocks to Eliminate them and help Fallen_Cat to lose weight.
  • How to playuse your mouse to select blocks and use WASD (If you are a mobile player, please use the keyboard in the left-up corner) on the keyboard to move blocks in the grid. Your goal is eliminate all fat blocks in the table. If you have no exercise block but fat blocks, you'll fail the stage.
Info for stage 1-2
  • HintYou can merge 2 exercise blocks together and their values will be stacked.
Info for stage 1-4
  • HintIf you don't take exercise or eat too much during a day, your body will get fatter.
    The value of exercise blocks and fat blocks WILL BE INCREASED BY 1 if you move them to an empty slot.
Info for stage 1-5
  • HintTry to use less clicks and less moves to finish the stage. Then your score will get higher.
Info for stage 1-6
  • HintYou cannot merge 2 fat blocks together.
Info for stage 1-7
  • HintA "switch" refers to selecting a new block. Switches slightly contributes to your score!
Info for stage 1-9
  • HintTry to form a long path of blocks and combine blocks along the path at once. This will help reduce the number of clicks.
Info for stage 2-1
  • New blockBarrier Block
    A kind of block which cannot be moved or merged with any other blocks. It's useless.
Info for stage 3-1
  • New blockMultiply Block
    A kind of block which can multiply the value of exercise blocks or fat blocks when they moved onto it. But itself cannot be moved to any place.
Info for stage 3-4
  • New blockDelete Block
    A kind of block which can eliminate exercise blocks or fat blocks when they moved onto it. But itself cannot be moved to any place.
  • HintYou can reach a weight below 45 kilograms in some stages. Try to reach lower weight in these stage and get a higher score!
Info for stage 3-9
  • HintThe modder's brain went spasmodic on doing this one.
Info for stage 5-1
  • New blockXOR Block
    When you move your exercise blocks or fat blocks onto it, their absolute value will be XORed (^)! But itself cannot be moved to any place.
  • HintIf you don't know what XOR is, you can click here to read.
Info for stage 5-2
  • HintYou will need some strategy when using this block...
Info for stage 5-7
  • HintThe placing of tiles can sometimes (mathematically) prevent you from making use of the XOR tile in the way you want. Think of other ways.
Info for stage 6-1
  • New blockSlide Block
    When you move your exercise blocks or fat blocks onto it, they will slide to the direction of the slide block! But itself cannot be moved to any place, and it can only be used once.
Info for stage 6-2
  • HintSlide blocks will slide your block onto other blocks... unless it is unable to. Try it out!
  • HintThis is a perfect example of why the solution with the least weight is not always the best.
Info for stage 6-3

Info for stage 6-6
  • HintYou can prevent the fat tile from increasing by blocking a fat tile in front of where it is going to go!
